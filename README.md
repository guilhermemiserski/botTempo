# BotTempo


BotTempo é um bot desenvolvido em python que responde qual a previsão do tempo no dia de hoje.


### Instalação

Instale as dependências e de deploy no [Heroku](https://dashboard.heroku.com/)!

```sh
cd bottempo
pip install -r requirements.txt
```
